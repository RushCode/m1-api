### About this class

This is a PHP7 API implementation for [M1 Messenger](https://play.google.com/store/apps/details?id=smile.m1project)

## Installation

Add the following to your composer.json:

```json
{
  "require": {
    "leocata/m1-api": "~1.0"
  }
}
```
